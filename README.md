Welcome to KhaneWala, a food ordering website built using the MERN (MongoDB, Express, React, Node.js) stack. This application allows users to browse a variety of delicious dishes, place orders, and enjoy a seamless food ordering experience.

## Features

- **User Authentication**: Users can sign up and log in to their accounts.
- **Browse Menu**: Users can browse a wide range of food items available for ordering.
- **Search and Filters**: Easily search for specific dishes and apply filters based on preferences.
- **Add to Cart**: Users can add items to their cart for ordering.
- **Place Orders**: Place orders from the cart with delivery address and payment details.
- **Order History**: Users can view their order history.
- **Admin Dashboard**: Admin users have access to manage dishes, orders, and user information.
- **Responsive Design**: The website is responsive, ensuring a seamless experience across devices.

## Getting Started

Follow these instructions to set up and run the project locally on your machine.

### Prerequisites

- Node.js and npm installed on your machine
- MongoDB server up and running

### Installation

1. Clone the repository: `git clone https://github.com/your-username/KhaneWala.git`
2. Navigate to the project directory: `cd KhaneWala`
3. Install frontend dependencies: `cd client && npm install`
4. Install backend dependencies: `cd ../server && npm install`
5. Create a `.env` file in the `server` directory and add your environment variables.

Example `.env` file:

```plaintext
MONGODB_URI=your-mongodb-connection-string
JWT_SECRET=your-secret-key
```

6. Run the frontend and backend concurrently: `npm run dev`

The project should now be running locally. Open your browser and navigate to `http://localhost:3000` to access KhaneWala.

## Usage

1. Sign up or log in to your account.
2. Browse the menu, add dishes to your cart.
3. View your cart and proceed to checkout.
4. Enter delivery details and complete the order.
5. Admins can access the admin dashboard by visiting `/admin`.
